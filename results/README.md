# Results Summary

This folder contains the outputs generated during the analysis phases of the Chicago Vanguard project.

## Files

### rdd_analysis_results.txt
Contains distributed analytics generated using Apache Spark RDD operations, including:
- Crime frequency analysis
- District statistics
- Crime type distribution
- Time-based crime patterns

### sql_analysis_results.txt
Contains Spark SQL query outputs, including:
- Crime distribution by district
- Arrest rate analysis
- Time-of-day patterns
- Weekday vs weekend analysis

### kmeans_clustering_results.txt
Contains MLlib K-Means clustering results:
- Selected features
- Silhouette evaluation
- Cluster distribution
- Final model performance
