# Source Code

This folder contains the implementation of the Chicago Vanguard Big Data Crime Analytics project.

## Files

### 01_DataPreprocessing.scala
Data cleaning, feature engineering, and dataset preparation.

### 02_RDDOperations.scala
Apache Spark RDD analysis including crime statistics, district analysis, and time-based crime patterns.

### 03_SQLOperations.scala
Spark SQL queries used for crime aggregation, arrest rate analysis, and district-level insights.

### 04_MachineLearning.scala
Spark MLlib implementation using K-Means clustering and silhouette score evaluation.

## Technologies

- Scala
- Apache Spark
- Spark SQL
- Spark MLlib
- K-Means Clustering
